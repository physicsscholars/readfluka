ReadFLUKA

Introduction

This aim of this project is to provide a user-friendly C++ interface for reading the binary output produced by different FLUKA estimators and for drawing the obtained values with such tools as ROOT or Gunplot.

Installation

  • To install ReadFLUKA from source you first have either to get the tar file containing the source or to use the Subversion repository to get the latest verstion.
    The tar file can be downloaded from here: http://code.google.com/p/readfluka.
    Alternatively, you can checkout the head (development version) of the source: svn co http://code.google.com/p/readfluka/trunk readfluka
  • Run the file env.sh from the libReadFluka folder. This bash script does nothing but exporting two local variables:
    export READFLUKA_SYS=/path/to/libReadFluka
    export LD_LIBRARY_PATH=$READFLUKA_SYS:$LD_LIBRARY_PATH
  • Go to $READFLUKA_SYS and type make.

    • Information on third party software

    ROOT

    If you are going to use the library with ROOT, please be sure that you have the ROOTSYS variable correctly defined before you install ReadFLUKA.
